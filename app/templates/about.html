About

Quality estimation of restaurant reviews using the Yelp Open Dataset.

Dataset

The models are trained on a large subset of the Yelp Open Dataset, specifically filtered for "Restaurants". To ensure balanced learning, the classes were harmonized and cleaned, resulting in a training corpus of over 1.14 million reviews.

Total Raw Reviews~6.9 Million
Filtered Restaurants~4.7 Million
Final Cleaned Corpus1,139,836
Deep Learning Subset100,000
Models
Model Type Architecture / Input Strength
Logistic Regression Classical ML TF-IDF (50k features) + 8 Numeric Fastest inference, highly interpretable.
SVM (LinearSVC) Classical ML TF-IDF + Numeric (Calibrated) Excellent margin separation.
LSTM Deep Learning Embedding (128) → LSTM (64) Captures sequential long-term dependencies.
BiLSTM Deep Learning Embedding → Bidirectional LSTM Understands both past and future context.
CNN+LSTM Deep Learning Conv1D → MaxPooling → LSTM Extracts local n-gram features efficiently.
Quality Classes
Poor (Kötü)
1 - 2 Stars
Negative experiences, complaints, bad service, or poor food quality.
Average (Orta)
3 Stars
Neutral feelings, mixed reviews, "okay" but not outstanding.
Good (İyi)
4 - 5 Stars
Positive feedback, recommendations, excellent food and service.
How to Run
1.
Install dependencies: pip install -r requirements.txt
2.
Run data preparation (Notebook 1 & 3): python data_preparation.py
3.
Train models (Notebook 4 & 5): Extract TF-IDF and train LR, SVM, LSTM.
4.
Start UI: cd app && python app.py
